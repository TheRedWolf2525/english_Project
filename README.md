## english_Project

# Installation :

**Environnement virtuel**
Créer environnement :
`python3 -m venv env`

Activer environnement :
`env/bin/activate`

Maj librairies : 
`pip freeze > requirements.txt`

Installer librairies nécessaires :
`pip install -r requirements.txt`

Désactiver environnement :
`deactivate`
